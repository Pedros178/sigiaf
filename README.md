# SIGIAF - Sistema de Gestión



Sistema con backend en Node.js y base de datos PostgreSQL.
Instalar:

* Node.js
* Express
* PostgreSQL


 Instalación

### 1. Clonar repositorio

git clone https://github.com/TU-USUARIO/sigiaf

### 2. Backend

cd backend
npm install

### 3. Configurar variables de entorno

Crear archivo `.env` basado en `.env.example` con tus propias credenciales 
 
### 4. Ejecutar backend

npm run dev

## 🗄️ Base de datos

Ejecutar el script SQL:

psql -U postgres -f database/SIGIAF.sql

##  Notas

* Asegúrate de tener PostgreSQL activo
* Configura correctamente el archivo `.env`

## 👨‍💻Frontend


cd frontend
npm startnpm install
npm start
